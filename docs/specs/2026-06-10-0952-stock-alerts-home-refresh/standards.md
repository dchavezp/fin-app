# Standards for Stock Alerts Home Refresh

The following standards apply to this work.

## native/expo-router

Use nested stack routes under the Alerts tab for list/create/edit screens.

## native/hooks-architecture

Keep shared alert state in a focused hook/provider and keep screen components declarative.

## native/ui-constraints

Use the existing React Native + StyleSheet visual language already used by the app.

## native/performance

Keep the preview small and render only the first four alerts on Home.
