# Product Mission

## Problem

Develop a full-stack application (React Native + Node Backend) for showing stock market data in real time using Finnhub Stock APIs. Users need a mobile-first way to track stocks, set price alerts, and receive push notifications when their targets are hit.

## Target Users

Trading enthusiasts and investors who want real-time stock data, price alerts, and portfolio visualization on mobile.

## Solution

A mobile app with:
- Email/password authentication
- Stock price alert creation
- Real-time stock listings via Finnhub API
- Charts/graphs for all stocks
- Firebase Cloud Messaging push notifications when alert prices are hit
- Docker-based deployment for the backend
